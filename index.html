Hauptprojekt

Bachelorarbeit

BACHELORKOLLOQUIUM · 03.07.2026 · NOTE 1,3

Mensch-in-der-Schleife-Agenten in der Pflegerobotik

Eine agentische Multi-Pipeline-Architektur, die Pflegedokumentation direkt auf einem realen Pepper-Roboter verarbeitet — mit einem 4-Bit-quantisierten Vision Routing Agent zur Frame-Selektion und einem interaktiven Human-in-the-Loop-System zur Fehlerkorrektur.

  • Vision Routing Agent: leichtgewichtiges 4-Bit-VLM mit Few-Shot-Prompting zur Frame-Auswahl — In-Memory-Verarbeitung ohne Datenpersistenz (DSGVO-konform)
  • Vergleichende Evaluation dreier Extraktionspipelines: OCR+LLM-Baseline, VLM Few-Shot, Hybrid — über Qwen3-VL und Pixtral
  • Zweistufige Evaluation: Transkriptionsgüte (ROUGE, BERTScore) und nachgelagerte Extraktionsqualität, getestet auf synthetischem N=48-Datensatz
  • Human-in-the-Loop-Feedbacksystem mit TTS-Sprachausgabe, produktiv auf Pepper-Hardware (Android, QiSDK)
